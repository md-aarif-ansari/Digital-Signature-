# 📄 Digital Signature Management System

> A modern full-stack platform for secure document management, digital signing, and workflow tracking.

---

## 🚀 Overview

The Digital Signature Management System is a secure web application that enables users to upload, manage, track, and digitally sign documents through an intuitive dashboard.

The platform provides secure authentication, role-based access control, document lifecycle management, audit logging, and signature workflow tracking.

This project demonstrates enterprise-level full-stack development using React, Spring Boot, JWT Authentication, and MySQL.

## ⚙️ Installation

### Backend

```bash
cd backend
mvn spring-boot:run
```

Backend URL:

```text
http://localhost:8080
```

Swagger:

```text
http://localhost:8080/swagger-ui/index.html
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend URL:

```text
http://localhost:5173
```
